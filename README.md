# Instagram 中文情感分析

## 🛠 使用資料
- 資料集：
  - 撰寫爬蟲程式抓取 Instagram 文章之留言
- 資料前處理：
  - 正則表示式過濾多餘內容

## 🛠 使用技術
- 自然語言處理（NLP）：
  - 中文斷詞（jieba）
  - 停用詞（Stopwords）過濾
  - 文字前處理（正則表達式清洗）
- 機器學習：
  - Logistic Regression（二元分類）
  - Sigmoid 函數
  - 梯度下降法（自行實作，非套件）
- 特徵工程：
  - 詞頻特徵（Word Frequency）
  - 正負情緒詞彙加權

## 🛠 評估方法
- 模型評估：
  - 模型將輸出 0 或 1 作為預測值，透過比對預測值與實際標籤計算正確率。測試集上的正確率達91%
