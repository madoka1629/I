# Instagram 中文情感分析

專案簡介：
本專案預期透過模型判斷民眾對於貼文的好感度。 
－本方法透過爬蟲蒐集約十萬筆 Instagram 貼文留言，並將留言以人工分類成正負面傾向。
－利用斷詞套件將所有留言進行斷詞，並利用斷出來的單詞建立字典，同時統計單詞的出現頻率。  
－字典建立後，本方法將欲預測的留言斷成單詞，將單詞對應字典中的出現頻率進行特徵處理。 
－將處理完成的特徵作為輸入，並使用 Logistic Regression 進行判斷正負面傾向，模型準確率約為 85%。

## 🛠 使用資料
- 資料集：
  - 撰寫爬蟲程式抓取 Instagram 文章之留言
- 資料前處理：
  - 正則表示式過濾多餘內容

## 🛠 使用技術
- 自然語言處理（NLP）：
  - 中文斷詞（jieba）
  - 停用詞（Stopwords）過濾
  - 文字前處理（正則表達式清洗）
- 機器學習：
  - Logistic Regression（二元分類）
  - Sigmoid 函數
  - 梯度下降法（自行實作，非套件）
- 特徵工程：
  - 詞頻特徵（Word Frequency）
  - 正負情緒詞彙加權

## 🛠 評估方法
- 模型評估：
  - 模型將輸出 0 或 1 作為預測值，透過比對預測值與實際標籤計算正確率。測試集上的正確率達 85%
